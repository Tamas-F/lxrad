LXRAD Layout HELP

The menu commands are:
File
New Project Open project or create new project
New Window Add window to project
Save Save project
Preferences Preferences apps (Browser, Editor and Debuger)
Exit Exit layout
Edit
Options Compiler Options
Clear Clear window
Delete Control Delete selected object
Include Files Include files for link (*.o, *.a)
Project
Run Run project
Build Build project
Clean Clean project
Edit Source Edit Source code of current window
Debug Run Debuger
Windows
Windowx Select window x for edition
